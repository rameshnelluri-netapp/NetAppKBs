1. What is NAS Backup?

NAS backup protects shared file storage such as:

  • NetApp ONTAP
  • Dell PowerScale / Isilon
  • Windows File Servers
  • Linux NFS Servers
  • SMB Shares
Rubrik performs agentless backup of NAS data.
